Help & FAQ

Everything you need to know to play Bingo Multiplayer

🚀 Getting Started

How do I join a game?

You need a room code from the person who created the game (the host). On the home screen, enter your name and the 6-character room code, then tap Join Game →. No account required!

Do I need to create an account?

No. You can play as a guest — just pick a fun name (or use the 🎲 random button) and enter the room code. If you want to create your own rooms and host games, you need to sign in with Google or an email/password.

What is a room code?

A room code is a 6-character code (letters and numbers, e.g. BK4R7Z) that the host shares with friends to invite them into a private game. Rooms are invite-only — nobody can stumble into your game without the code.

How do I create a room and invite friends?

1

Sign in (Google or email). The lobby appears.

2

Choose your Game Mode, Win Rule, and (for Turn/Classic) how to pick the first caller.

3

Tap + Create Room. You'll enter the game waiting room.

4

Share the room code shown in the game header (green monospace code) with your friends — via WhatsApp, text, or chat.

5

Once everyone has joined, tap Start.

💡 Friends don't need an account — they just enter the code on the guest screen.

🎮 Game Modes

⏱ Auto Auto Mode

Numbers are called automatically every 8 seconds. No one needs to do anything — just watch and mark your card. Great for large groups or casual play.

👆 Turn Turn Mode

Players take turns calling numbers in a round-robin rotation. When it's your turn, a 📣 Call! button appears. The system picks a random number for you. If you don't call in 15 seconds, the host auto-calls for you.

🎯 Classic Classic Mode

Like a real bingo hall. The caller picks a number by tapping it on their own board (blue glowing cells). Numbers 1–25 are used (no B-I-N-G-O columns). Takes turns round-robin style.

🥇 Who Calls First? (Turn & Classic modes)

Random

The app randomly shuffles all players and assigns a calling order.

Rock Paper Scissors (RPS)

All players choose 🪨 Rock, 📄 Paper, or ✂️ Scissors (15-second window). The winner calls first; losers follow. On a tie, everyone plays again. If you don't pick in time, a random choice is made for you.

💡 Playing Tips

What are the gold glowing numbers on my board?

Gold-glowing cells show numbers that have been called but you haven't marked yet. Tap them to mark! You can toggle this highlight on/off with the 💡 button in the game header.

What are the blue glowing numbers? (Classic mode)

Blue-glowing cells appear on the caller's board in Classic mode. They are numbers that haven't been called yet — tap one to call it as the next number!

Can I mark a number that hasn't been called?

No. If you tap a number that hasn't been called, the cell briefly flashes red to let you know. Only called numbers can be marked.

How do I know whose turn it is?

The top bar shows the current caller's name. When it's your turn, it says "Your turn to call!" in gold. The timer counts down how long you have.

🔧 Troubleshooting

I entered the room code but got "Room not found"

  • Double-check the code — it's case-insensitive but must be the right 6 characters.
  • Room codes use only unambiguous characters (no zero/O, no one/I/L confusion).
  • Ask the host to confirm the code shown in their game header.
  • The room must still be in "waiting" status — if the game started, you can't join.

The game says "That room already started"

Rooms only accept new players before the host taps Start. Ask the host to create a new room for you to join before the game begins.

I refreshed the page and lost my game

⚠️ Guests: refreshing the page ends your session. You'll need to get a new code and rejoin (if the game hasn't started). Registered users stay logged in after a refresh but may need to re-navigate to the game.

Other players are missing from the player list

The player list updates in real time. If someone's name is missing, they may not have joined yet, or there may be a short sync delay. Wait a few seconds and it should update.

The game is stuck / Start button doesn't work

  • Only the host (room creator) can start the game.
  • Make sure at least one player has joined.
  • If there's a connection issue, try leaving and re-creating the room.

What happens if the host leaves?

If the host leaves mid-game, the game will continue until a winner is found in Auto mode (numbers stop being called), or stall in Turn/Classic mode (no one can advance the turn). For best results, the host should stay until the game ends.

📱 Device Support

Bingo Multiplayer works in any modern browser on phones, tablets, and desktops. No app download required. Chrome, Safari, Firefox, and Edge are all supported. JavaScript must be enabled.
